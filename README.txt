== Description ==

Free forever! Show attractive call button on website with ring ring animation. The call button immediately displays the same as the call button of Coccoc.

= for Vietnamese =
Đây là Plugin hoàn toàn miễn phí cung cấp chức năng hiển thị nút Gọi ngay động như Alo Cốc cốc.

* Ưu điểm:
1. Dễ dàng cài đặt, phù hợp với mọi Giao diện và mọi phiên bản Wordpress
2. Gọn nhẹ, dễ dàng tùy chỉnh theo cách riêng, không xung đột với các plugin khác
3. Hiển thị đẹp mắt, chuyên nghiếp giúp nâng cao tương tác với người dùng website

* Nhược điểm:
Trong tương lai chúng tôi sẽ cải thiện hơn phần tùy chỉnh của Plugin để các bạn có thể tùy biến tùy thích theo ý bản thân.

== Installation ==


1. Upload `callnow_pht.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Setting -> Call Now Coccoc and put your number to begin
